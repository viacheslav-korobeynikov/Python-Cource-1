# Форматирование строк в Python
Форматирование строк — это способ вставки переменных в строку в Python. Они используются для форматирования строк и получения более удобочитаемых выводов. Существует несколько способов форматирования строк в Python:

## Интерполяция строк (f-строки)
Введенные в Python 3.6, f-строки — это новый способ форматирования строк в Python. Они начинаются с 'f' и используют фигурные скобки {}, чтобы заключить переменные, которые будут отформатированы. Например:
```python
name = "John"
age = 30
print(f"My name is {name} and I am {age} years old.")
```
Это выведет:
```python
My name is John and I am 30 years old.
```
## str.format()
Это еще один способ форматирования строк в Python. Он использует фигурные скобки {} в качестве заполнителей для переменных, которые передаются в метод format(). Например:
```python
name = "John"
age = 50
print("My name is {} and I am {} years old.".format(name, age))
```
Это выведет:
```python
My name is John and I am 50 years old.
```
## % Оператор
Это один из старейших способов форматирования строк в Python. Он использует оператор % для замены переменных в строке. Например:
```python
name = "Johnathan"
age = 30
print("My name is %s and I am %d years old." % (name, age))
```
Это выведет:
```python
My name is Johnathan and I am 30 years old.
```
**“Меня зовут %s, и мне %d лет.“**: Это строка, которая включает спецификаторы формата:

%s: Это заполнитель для строки.
%d: Это заполнитель для целого числа.
% (имя, возраст): Это кортеж, содержащий переменные имя и возраст. Значения этих переменных заменят заполнители в строке.

Каждый из этих методов имеет свои преимущества и области применения. Однако f-строки обычно считаются самым современным и предпочтительным способом форматирования строк в Python благодаря своей читаемости и производительности.
## Дополнительные возможности
F-строки также могут оценивать выражения внутри фигурных скобок, что может быть очень полезно. Например:
```python
x = 10
y = 20
print(f"The sum of x and y is {x+y}.")
```
Это выведет:
```python
The sum of x and y is 30.
```
## Сырая строка (r-строка)
В Python сырые строки являются мощным инструментом для работы с текстовыми данными, особенно при работе с символами экранирования. При добавлении префикса ‘r’ к строковому литералу Python обрабатывает строку как сырую, что означает, что он интерпретирует обратные слеши как буквальные символы, а не как последовательности экранирования.

Рассмотрим следующие примеры обычной строки и сырой строки
**Обычная строка:**
```python
regular_string = "C:\new_folder\file.txt"
print("Regular String:", regular_string)
```
Это выведет:
```python
Regular String:  C:
ew_folderile.txt
```
В обычной строке переменной regular_string обратные слэши (\n) интерпретируются как управляющие последовательности. Поэтому \n представляет собой символ новой строки, что приведет к неправильному представлению пути к файлу.
**Сырая строка:**
```python
raw_string = r"C:\new_folder\file.txt"
print("Raw String:", raw_string)
```
Это выведет:
```python
Raw String: C:\new_folder\file.txt
```
Однако в сырой строке raw_string обратные слэши рассматриваются как буквальные символы. Это означает, что \n не интерпретируется как символ новой строки, а воспринимается как два отдельных символа, \ и n. Следовательно, путь к файлу представлен точно так, как он выглядит.
